# 앱인토스(App in Toss) 게임 개발 가이드

## 1. 플랫폼 개요 및 개발 환경

- [cite_start]**개요**: 3,000만 토스 앱 사용자 대상의 미니앱 플랫폼으로, 별도 앱 설치 없이 실행됨[cite: 66, 67, 68].
- **게임 개발 환경 지원**:
  - [cite_start]HTML5 기반 개발 지원[cite: 353].
  - [cite_start]Unity 및 기타 게임 엔진 지원[cite: 354].
  - [cite_start]앱인토스 환경에서 실행 가능한 WebGL 형태 등으로 포팅 필요 (RN-Webview 지원)[cite: 365].

## 2. 최적화 필수 사항

- [cite_start]**용량 제한**: 초기 로딩 다운로드 리소스를 최소화해야 하며, **초기 번들 사이즈는 50MB 이하**를 권장함[cite: 367, 370].
- [cite_start]초기 로딩 중 유저 이탈을 막기 위해 패키지와 리소스를 분리하는 작업이 필요함[cite: 368, 369].

## 3. 제공되는 API / SDK (주요 기능)

- [cite_start]**회원 연계 및 로그인**: 토스 로그인, 토스 인증[cite: 170, 172, 422, 430].
- **수익화 (결제/광고)**:
  - [cite_start]토스페이 간편결제 SDK 지원[cite: 169, 425, 428, 575].
  - [cite_start]인앱 결제 (IAP)[cite: 167, 429, 439, 575].
  - [cite_start]인앱 광고 (IAA) - 토스 제공 애드몹 SDK 지원[cite: 166, 561, 562, 563].
- [cite_start]**마케팅 및 리텐션**: 기능성 푸시 알림 메시지 발송, 친구 초대 시 리워드를 제공하는 '게임 공유 리워드' 기능[cite: 424, 456, 517, 518].
- [cite_start]**디바이스 기능**: 위치 정보, 클립보드, 사진, 연락처 등 [cite: 173-178, 436-440].
